# Face Recognition Using DeepFace

## Overview
This project implements face detection, verification, and identification using DeepFace, built in Google Colab.

## Features
- Face detection
- Face verification (1:1)
- Face identification (1:N)
- Works with custom face database

## Technologies
- Python
- DeepFace
- Google Colab
- OpenCV

## How to Run
1. Upload images to face_db folder
2. Run notebook in Google Colab
3. Use find() for identification
4. Use verify() for comparison
